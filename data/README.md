# data/ — Pre-computed Data Directory

> [中文文档](README_zh.md)

This directory holds all pre-computed artefacts that AcadVex needs at runtime. **Large binary files are not committed to git** (see `.gitignore`). You must generate or download them before starting the application.

---

## Directory Layout

```
data/
├── embeddings/dblp/          ← FPGCL model output (git-ignored, *.pt)
│   ├── author_emb.pt         ← (4057 × 64) float32 tensor, one row per author
│   └── paper_emb.pt          ← (14328 × 64) float32 tensor, one row per paper
│
├── graph_stats/dblp/         ← Pre-computed graph metadata (git-ignored)
│   ├── author_meta.json      ← {id, name, community_id, keywords, ...} × 4057
│   ├── author_names.json     ← {author_id: name} lookup
│   ├── paper_meta.json       ← {id, title, venue_id, year, author_ids}
│   ├── paper_titles.json     ← {paper_id: title} lookup
│   ├── edges.json            ← [[src, dst], ...] co-authorship edges
│   ├── community_meta.json   ← {community_id: {name, color, ...}} × 4
│   └── venue_names.json      ← {venue_id: name} × 20
│
├── precomputed/              ← Small derived files (committed to git)
│   ├── centrality.json       ← {author_id: {degree, betweenness, closeness}}
│   └── community_topics.json ← {community_id: [top keywords]}
│
├── vector_stores/            ← RAG indices (git-ignored, large binaries)
│   ├── author_vectordb/      ← ChromaDB collection for author semantic search
│   └── author_bm25.pkl       ← BM25 index for keyword search
│
└── build_knowledge_base.py   ← Script to rebuild vector_stores/ from scratch
```

## How to Regenerate

### Embeddings (`embeddings/dblp/`)

Export from the trained FPGCL model in the companion repository:

```bash
# In the FPGCL repository
python export_for_app.py --dataset dblp --out /path/to/AcadVex/data/
```

Expected shapes: `author_emb.pt` → `(4057, 64)`, `paper_emb.pt` → `(14328, 64)`.

### Graph stats (`graph_stats/dblp/`)

Generated by the FPGCL export script above alongside the embeddings.

### Precomputed centrality and topics

```bash
cd AcadVex
python scripts/precompute/compute_centrality.py
python scripts/precompute/compute_topics.py
```

### Vector stores (`vector_stores/`)

```bash
python scripts/build_author_vectordb.py
```

This embeds all authors with `bge-small-zh-v1.5`, writes the ChromaDB collection, and builds the BM25 index. Requires `data/graph_stats/dblp/` to exist first.
